The purpose of this project is to analyze employee data and predict attrition using HR analytics techniques. The dataset includes employee information such as age, job role, salary, work experience, and job satisfaction. Data preprocessing, exploratory data analysis, and basic machine learning models are applied to identify key factors influencing attrition. The results help HR departments take proactive steps to reduce employee turnover and improve retention strategies.
Python – for data analysis and model building

Pandas – for data cleaning and manipulation
Matplotlib / Seaborn – for data visualization
Jupyter Notebook – for implementation and analysis
